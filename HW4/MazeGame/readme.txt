We have matched mazesolver game to match.h code.
The game is to solve hidden mazes. There are two algorithms that try to solve these mazes. The main idea of each algorithm is to put bookmarks in the right places and research the maze during the game to solve it. After each move player get feedback from gamemanager that helps him to find the right way to solve the maze. Each maze has max num of steps. There are couple of flags that give opportunities to user to set maze folder and output file folder.
